# 🍓 Sweets by Summeray

An interactive, animation-rich website for **Sweets by Summeray** — hand-dipped
chocolate covered strawberries, cake pops, crispy treats and chocolate pretzels.

## ✨ Highlights

- **Three.js hero** — a 3D backdrop of drifting pastel sprinkles, glossy bonbons
  and chocolate donuts with mouse parallax.
- **GSAP everywhere** — preloader, staggered hero text reveal, scroll-triggered
  section reveals, magnetic buttons, 3D-tilt treat cards, dual-direction
  marquees, and a custom trailing cursor.
- **The Dip Lab** — an interactive strawberry you can dunk into four chocolates
  with a squash-and-stretch animation.
- **Build a Box** — a video-game-like order builder:
  1. Pick your box (4 / 6 / 12 treats) with a HUD showing progress and total.
  2. Design each treat — type, chocolate dip, drizzle, topping — with a live
     SVG preview, then watch it *fly* into the next slot with sparkles.
  3. "Surprise me" auto-fills the box, confetti rains when it's full, and a
     wrapped summary generates a ready-to-send order email.
- All treat artwork is hand-built parameterized SVG — no photography needed.
- Pastel design system, `prefers-reduced-motion` support, responsive layout.

## 🛠 Tech

Vite · React · GSAP (ScrollTrigger) · Three.js · React Router

## 🚀 Run it

```bash
npm install
npm run dev      # local dev server
npm run build    # production build → dist/
npm run preview  # preview the production build
```
